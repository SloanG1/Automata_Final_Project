Standards and Practices:

File Names:
    File names must be lowercase and in snake case

Directory Names:
    Directory names must be lowercase and in snake case

Code Files:
    python code must be in proper OOP format with Data attributes, Init, Helpers, Getters, and Setters
    Imported code must be located at the top of each file

Driver File:
    Driver file must include a main method

Config File:
    Config file should only contain constant variables and should be accessible to all directories
